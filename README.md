# JetBrains IDEA Auto Zoom

Plugin for JetBrains IDEs (Rider/IntelliJ/...) to automatically set the zoom level based on the current display. Solves the issue of the UI being too small on widescreen desktops and too big on laptop screens.

## Features

- Automatically adjusts zoom level based on the current display
- Supports multiple JetBrains IDEs
- Remembers settings for displays and restores them when the IDE is shown on the particular screen
